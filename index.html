WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language HTML is the content of a web page.
Examples of HTML include headings, paragraphs, images, links, lists, etc.

Check out the HTML Tutorial w3schools.com.

CSS

CSS stands for Cascading Style Sheets. CSS is the style of a web page.
Examples of CSS include font styles and color, background color, borders,
dimensions, layout, etc.

Check out the CSS Tutorial on w3schools.com.

Learn JavaScript

JavaScript is the programming language of the web. JavaScript helps to make
web pages interactive and dynamic. Some commmon JavaScript tasks are:

  • add new HTML to the page,
  • change the existing content / style,
  • react to user events (mouse clicks, key presses, etc.)

Check out the JavaScript Tutorial on javascript.info.
